Plugin para colocar automáticamente el atributo ALT al subir imágenes, usando el nombre del archivo como base. Mejora la accesibilidad y el SEO de tus medios.

== Descripción ==

Este plugin añade automáticamente el atributo `alt` a las imágenes que se suben a la biblioteca de medios de WordPress, usando el nombre del archivo como base. También permite personalizar otros metadatos como el título, la leyenda y la descripción de forma automática si se desea (líneas comentadas en el código).

**Características principales:**

* Genera automáticamente el atributo `alt` basado en el nombre del archivo.
* Limpia caracteres como guiones y subrayados, y capitaliza palabras.
* Opcionalmente puede establecer título, leyenda y descripción.

== Instalación ==

1. Sube el plugin a la carpeta `/wp-content/plugins/images-alt-automatic`.
2. Actívalo desde el menú "Plugins" de WordPress.
3. El plugin comenzará a funcionar automáticamente al subir nuevas imágenes.

== FAQ ==

= ¿Funciona con imágenes ya subidas? =  
No. El plugin solo aplica los cambios al subir nuevas imágenes.

= ¿Puedo aplicar esto a imágenes existentes? =  
No de forma automática con este plugin, pero puedes hacerlo manualmente o con otro plugin complementario.
